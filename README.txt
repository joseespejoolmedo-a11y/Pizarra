# Pizarra PRO (GitHub Pages)
Descomprime y sube **index.html** a la raíz del repositorio en la rama `main`.

## Pasos rápidos
1. *Add file → Upload files* y arrastra **index.html** (descomprimido).
2. *Settings → Pages* → Source: **Deploy from a branch**, Branch: **main**, Folder: **/(root)**.
3. Abre `https://TU-USUARIO.github.io/Pizarra/?v=2`.

> GitHub **no** descomprime ZIP automáticamente.
